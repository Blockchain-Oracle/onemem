# onemem-sdk-python

Python SDK for OneMem using `pysui`.

**Publication note, 2026-06-18:** `onemem-sdk-python@0.2.0` is current on PyPI
after `pnpm registry:status --strict`. Re-run that command before making a
fresh public install claim.

Current API truth is the source under `onemem/`. The historical shared API spec
in `docs/05-our-architecture/02-sdks/shared-api-surface.md` may describe
planned TS/Python parity that is not fully implemented yet.
