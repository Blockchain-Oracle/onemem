# onemem-sdk-python

Python SDK for OneMem using `pysui`.

**Publication note, 2026-06-18:** this README describes repo-local
`onemem-sdk-python@0.2.0`. `pnpm registry:status` currently reports the package
as missing from PyPI, so package consumers should not claim public install proof
until registry status reports it current.

Current API truth is the source under `onemem/`. The historical shared API spec
in `docs/05-our-architecture/02-sdks/shared-api-surface.md` may describe
planned TS/Python parity that is not fully implemented yet.
