# OneMem

OneMem is **decentralized persistent memory for AI agents**. It gives any agent two things its runtime can't keep reliably across tools:

1. **Portable memory** - every memory is a Seal-encrypted blob on Walrus, access-controlled by an on-chain namespace. Portable across runtimes, decryptable only by delegate-key holders.
2. **Verifiable action history** - every tool / skill / MCP call can be recorded as a Merkle-chained `ActionCall` anchored on Sui. Anyone can replay it, and anyone can **verify** it - no login, no trust in us.

Built on the Walrus-track stack: **Sui** (anchoring) · **Walrus** (blob storage) · **Seal** (threshold encryption) · **MemWal** (memory relayer).

## The trust model

OneMem uses Seal's **`/manual`** flow: content is encrypted client-side before it touches the relayer, so **the relayer never sees plaintext**. The on-chain commit chain is public; the blob contents stay private. Verification walks the public chain — it proves *integrity*, not contents.
