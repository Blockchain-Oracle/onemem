I am building the brand identity for OneMem, a Sui Overflow 2026 hackathon project.

Context:
- Product name: OneMem
- Public website: onememe.xyz
- X handle: @OneMemAI
- Category: verifiable AI agent memory and action trace layer
- One-line positioning: Etherscan for AI agents
- Product summary: OneMem gives AI agents portable, encrypted, verifiable memory. Every memory is stored as a Seal-encrypted Walrus blob. Every action/tool call is a Merkle-chained attestation on Sui. Users can verify, replay, and share what an agent did across runtimes such as Claude Code, OpenClaw, Hermes, Cursor, Vercel AI SDK, OpenAI Agents SDK, CrewAI, LiveKit, and ElevenLabs.
- Core stack: Sui, Walrus, Seal, MemWal
- Hackathon track fit: Walrus / agentic systems / persistent AI memory
- Core promise: agents become more trustworthy when their memory and actions are durable, portable, and independently verifiable.
- Existing product copy:
  - "Stop trusting your AI agent. Verify it."
  - "Verifiable agent memory + trace, for every runtime."
  - "Every memory encrypted on Walrus, every action a Merkle-chained attestation on Sui."
  - "Verification turns the page green."

Existing visual DNA:
- Current app icon: dark rounded square with a cube-like memory/chain mark.
- Icon colors: charcoal `#171717`, verification lime `#d4ff5e`, violet `#8f7cff`.
- Brand token roles:
  - primary indigo/violet for memory and agent runtime identity
  - lime/green only for verification, decrypted content, successful proof moments
  - sea-blue only for on-chain/Sui explorer links
  - warm near-white paper surfaces for product/landing readability
  - dark charcoal/navy surfaces for cinematic social assets
- Typography direction: technical but polished. Display type can feel expressive and modern; body type should stay highly readable. Current product uses Bricolage Grotesque, Hanken Grotesk, and JetBrains Mono.

## First Motion Concept

Title: "Verification Turns The Page Green"

Storyboard:
1. Dark screen. A faint agent trace appears as scattered violet nodes.
2. Nodes connect into a Merkle chain, each call snapping into place with a soft tick.
3. Encrypted memory blocks float in from the side as dark cubes.
4. A Sui transaction hash line anchors the chain below.
5. OneMem mark draws itself: cube outline, then center trace line.
6. The verification pass sweeps across the chain in lime.
7. Final frame: OneMem logo, motto "Every memory. Every action. Proven.", `@OneMemAI`, and `onememe.xyz`.

Sound direction:
- Low, precise digital pulse.
- Soft confirmation ticks as trace nodes verify.
- Short warm lift at the green verification sweep.
- No hype-trailer boom; keep it premium and technical.
